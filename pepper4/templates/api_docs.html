API Documentation

All endpoints require an API key in the header: X-API-Key: your_api_key_here

Available Endpoints

  • GET /api/forms
    Query params: type (ai_generated/external), active (true/false)
  • GET /api/forms/<id>
    Get detailed information about a specific form
  • GET /api/stats
    Get overall statistics about forms

Example Usage

curl -H "X-API-Key: your_api_key_here" https://your-domain.com/api/forms
